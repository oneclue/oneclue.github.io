Windows下nvm的安装及基本使用

2020-05-28 other 0 评论 字数统计: 423(字) 阅读时长: 1(分)

 nvm（node version management）是 node 版本管理工具，其主要功能是可以在本地安装多个版本的 nodejs，且可切换版本。如果读者使用的是 macOS 或 linux 可以参考 nvm-sh/nvm 项目，如果是 windows 的话则参考 coreybutler/nvm-windows 项目。这是两个完全独立的项目。由于笔者本人使用 windows，所以这里下载安装以后者为例。

nvm-windows 下载和安装

github 地址：https://github.com/coreybutler/nvm-windows/releases

 下载 nvm-noinstall.zip 这是免安装版本，下载完后解压到本地的一个目录，读者可根据自己的安装目录来决定自己的安装位置，解压后看到 elevate.cmd、elevate.vbs、install.cmd、LICENSE、nvm.exe 。这时我们需要配置一下环境变量

 如果是添加系统级的环境变量添加完后需要重启一下，如果是用户级的话就不用。我们要设置两个环境变量一个是 NVM_HOME 指向 nvm 安装的目录，另一个是 NVM_SYMLINK 指向一个快捷方式如下图，我们在切换 node 版本时，目标会自动的改变为对应信息。

 配置好环境变量后，到 nvm 安装目录下新建一个 settings.txt 文本文件

 然后打开并编辑 setting.txt

 打开终端工具输入 nvm 出现 nvm 的相关信息，这样就安装成功了。
